## 🚀 Featured Projects

### 🔮 [Arcana Mística](https://github.com/AndyV01/arcana-mystica) - [LIVE DEMO](https://arcana-mystica.vercel.app)

Tarot app with a built-in **real multi-agent AI system**. Interpretations are generated by 6 specialized agents that plan, execute, and self-correct using **Groq API (Llama 3.3)** in a Vercel Serverless Function.

`React 18` `Vite 5` `Multi-Agent AI` `Groq API` `Serverless` `Node.js` `i18n EN/ES`

- 🤖 Orchestrator + 6 agents (Planner · Prompt · Critic · Hook · Memory · Rag ) with shared context
- 🔮 78 cards, 4 spreads, 3D flip, numerology and personalized zodiac
- 🌟 Letter of the Day · Weekly Horoscope · Persistent Diary · Sharing Card
- ⚡ Vercel Serverless Function with secure API key in environment variables

### 🤖 [Job Assistant AI](https://github.com/AndyV01/job-assistant-ai)
Multi-agent system with real LLM tool calling (Groq + LangChain), LangGraph StateGraph with persistent memory and retry logic, CV Optimizer with real RAG (FAISS + Embeddings) and real data via Adzuna API. Backend Python + FastAPI, Frontend React, deployed in Railway and Vercel.

 `Python` `FastAPI` `LangChain` `LagGraph` `Groq API` `RAG` `Adzuna API` `React` `Vite` `TailwindCSS`

🔗 [Repo](https://github.com/AndyV01/job-assistant-ai) | 🌐 [Demo](https://job-assistant-ai-tzle.vercel.app/)
